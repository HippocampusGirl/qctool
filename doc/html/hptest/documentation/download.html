HPTEST
Software to test for genetic association between hosts and pathogens.

Obtaining HPTEST

HPTEST is currently available as source code. The source code to HPTEST is available as part of the QCTOOL package, which can be obtained at https://code.enkre.net/qctool. QCTOOL is currently maintained as a fossil repository. To obtain source code, use the fossil command-line client, or alternatively download a tarball from this table. (Note that this currently generates a tarball on the fly, which may take a while).

v2.1.9https://code.enkre.net/qctool/zip/v2.1.9/qctool.tgz

Versioning scheme

As of version 2.1, QCTOOL will follow a versioning scheme whereby odd minor numbers (2.1.x etc.) represent development work while even numbers (2.0.x) represent release versions. The current latest release versions are 2.0 version.

Compiling

Having downloaded the source code (and extracted it if necessary), a basic compilation sequence would then be: 

$ cd qctool
$ ./waf-1.5.18 configure
$ ./waf-1.5.18

If all is successful, this produces an executable ./build/release/qctool_v2.0-<version>

You can check qctool is working by running it: 

$ ./build/release/qctool_v2.0-<version> -help

which should print out a list of options.

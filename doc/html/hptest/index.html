HPTEST
Software to test for genetic association between hosts and pathogens.

HPTEST is a command-line utility program that implements association tests between all pairs of variants in two sets of genotypes on the same samples. The motivating use case is to test for association between genetic variants in a host and pathogen.

HPTEST currently implements a logistic regression method in which the host (or pathogen) genotypes are included as predictors and the pathogen (or host) genotypes are included as the outcome. Named covariates can also be included. Imputed data is also supported for the predictor variable; the SNPTEST model is used to sum over the imputed genotype probability distribution.

# Introduction # HPTEST is a command-line utility program that implements association tests between all pairs of variants in two sets of genotypes on the same samples. The motivating use case is to test for association between genetic variants in a host and pathogen, but HPTEST also has other possible use-cases, such as estimating LD between variants while controlling for confounding variables. HPTEST currently implements a logistic regression method in one set of genotypes (e.g. host genotypes) are included as predictors and a second set (e.g. pathogen genotypes) are included as the outcome. Named covariates can also be included. For the predictor variable, imputed genotypes are supported for the predictor variable; in this case the SNPTEST model is used to sum over the imputed genotype probability distribution. For an introduction to running HPTEST, please see the Basic usage page. # References # HPTEST was developed to test for association between human and malaria genotypes in samples from the MalariaGEN CP1 study.
